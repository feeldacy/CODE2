PERBEDAAN ATTRIBUT DISABLE DAN READONLY

ATTRIBUTE DISABLED


Atribut Disable ketika digunakan maka elemen yang memiliki atribut tersebut akan menjadi nonaktif atau tidak bisa diapa-apakan.

ATTRIBUT READONLY


Atribut readonly ketika digunakan pada sebuah elemen maka elemen tersebut hanya bisa dibaca saja tanpa bisa diesit.

Perbedaan


Perbedaan dari kedua atribut itu adalah jika atribut readonly akan tetap mengirimkan nilai ke server sedangkan disabled tidak. Disabled digunakan untuk menyembunyikan fungsionalitas sebuah elemen, sedangkan readonly hanya mengurangi/mencegah aktivitas pengeditan.
